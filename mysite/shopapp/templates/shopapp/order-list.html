This HTML template is used to display a list of orders in the shop application. Description: - It defines a standard HTML document structure with the DOCTYPE declaration and sets the language to English. - It sets the title of the page to "Orders". - Inside the body of the page, it displays a heading "Orders:". - It iterates over each order in the provided context variable 'order' and displays details for each order, including delivery address, promocode, and color. - For each order, it creates a hyperlink to view the order details using the order's primary key (pk). - It includes links to create a new order and to navigate to the products list page. Note: The context variable 'order' should be passed to this template containing a list of order objects. Orders
