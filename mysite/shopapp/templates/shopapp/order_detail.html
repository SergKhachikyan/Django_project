This HTML template is used to display details of a single order in the shop application. Description: - It extends the base template 'shopapp/base.html'. - It defines a block for the title, which includes the order number (pk). - Within the body block, it displays the order details including delivery address, promocode, and color. - It includes hyperlinks to navigate back to the orders list and to update the current order. Note: The context variable 'object' should be passed to this template containing the order object to be displayed. {% block title %} Order #{{ object.pk }} {% endblock %} {% block body %}
